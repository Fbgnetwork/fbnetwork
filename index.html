What is FBG Network?

FreeBuild Network is a Minecraft server for Minecraft Java Edition.  

User friendly

Up to date

Fast support

How to join

Here you can see, how you can join on the Minecraft server.

1

Add the server

Go to Multiplayer in Minecraft, then Add Server
2

Enter server adress

Enter FBGNetwork.mc.gg in the server address field and click "Done".
3

You're done!

Now you can play on FBGNetwork.mc.gg with other players!
